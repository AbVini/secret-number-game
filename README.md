# Jogo do Número Secreto 🤔

Um simples jogo de adivinhação de números em JavaScript.

## Descrição😱

Este é um jogo onde o jogador deve adivinhar um número secreto gerado aleatoriamente. O jogador recebe dicas sobre se o número secreto é maior ou menor do que a tentativa atual.

## Funcionalidades☝🤓

- Geração aleatória de número secreto.
- Dicas sobre se o número secreto é maior ou menor.
- Contagem de tentativas.
- Reinício do jogo.
- Utilização de Arrays para armazenar numeros ja sorteados

## Instruções👨‍🏫

1. Abra o arquivo `index.html` no seu navegador.
2. Escolha um número entre 1 e 10.
3. Receba dicas sobre se o número secreto é maior ou menor.
4. Continue até acertar o número ou reinicie o jogo.
